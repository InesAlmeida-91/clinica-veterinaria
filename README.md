# Clínica Veterinária 🐾

## Requisitos
- XAMPP / MariaDB
- Python
- mysql-connector-python

## Instalar dependência
pip install mysql-connector-python

## Base de dados
Importar o ficheiro SQL no phpMyAdmin

## Executar
python gestao_recepcao.py
python gestao_consultas.py

## Funcionalidades
- Gestão de donos e animais
- Consultas
- Tratamentos
- Histórico clínico
